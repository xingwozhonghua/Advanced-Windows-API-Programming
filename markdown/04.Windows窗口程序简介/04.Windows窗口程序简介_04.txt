技术	简介	语言
Win32 API	Windows平台底层API、效率高、难J戴高	C+ +

WPF框架	WPF (Windows Presentation Foundation)是微软推出的 基于Windows的用户界面框架，基于DirectX 9/10技术 ,使fflXAML设th属于.NET Framework的一分。	C#
Windows Forms	微软的.NET Framework框架的图形用户界面部分，该组件 通过将现有的Windows API (Win32 API)封装为托管代码 提供了财Windows本地(native)组件的访问方式<>	C#、VB
